# Student Management System with Data Persistence

This project is a simple system for managing basic student information.

The project is structured modularly to ensure readability, scalability, and proper handling of errors and exceptions.

---

## 🚀 Features

* **Register new students.

* **View the student list.

* **Search for a student by criteria (Ej: ID or name).

* **Update student information.

* **Delete students.

### How it works

* The program displays a menu with several options.
* The user selects an option by entering a number.
* The system executes the corresponding operation.
* Data is managed during program execution.

## 🛠️ Requirements and Installation

1. Ensure you have Python 3.x installed on your system.

2. Clone this repository or download the files to a local folder.

3. (Optional) Save your system flowchart in the project root folder in `PNG` or `PDF` format.
